## MSSA Module 5 - Assignment 5.3: Placing Flowers & Climbing Stairs

## Description
This project contains two algorithm challenges completed as part of the MSSA CAD Program:

- **Placing Flowers** - Determines if flowers can be placed in a flowerbed without violating the no-adjacent-flowers rule.
- **Climbing Stairs** - Calculates the number of distinct ways to climb to the top of a staircase, taking 1 or 2 steps at a time.

## Technologies Used
- C# (.NET 8.0)
- Visual Studio 2022
- Console Application

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Build and run the project
